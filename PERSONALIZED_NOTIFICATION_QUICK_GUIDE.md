# Personalized Notification - Quick Guide

## What's New

✨ **Test personalized notifications from menstruation screen**

## How to Use

### Step 1: Open Menstruation Screen
- Navigate to menstruation tracker

### Step 2: Click Notification Icon
- Top right of screen (🔔)
- Next to wallet and voucher buttons

### Step 3: Select "Personalized"
- Dialog shows 4 options
- Tap "Personalized"
- Subtitle: "Based on your health data"

### Step 4: Notification Sent
- Local notification appears
- WhatsApp message sent (if phone saved)
- Success message shown

| Issue | Solution |
|-------|----------|

| Error message | Try again or restart app |

## What Happens Behind the Scenes

1. Gets user's menstruation logs
2. Calculates current cycle day
3. Determines cycle phase
4. Generates personalized message
5. Sends local notification
6. Sends WhatsApp (if phone saved)
7. Shows success message

## Integration

- Uses existing notification system
- Integrates with phone number feature
- Works with Twilio WhatsApp
- Uses user's cycle data
