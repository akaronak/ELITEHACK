🚀 QUICK START (3 STEPS - 5 MINUTES)

Step 1: Add Twilio Auth Token
  → Open server/.env
  → Find: TWILIO_AUTH_TOKEN=your_auth_token_here
  → Replace with your actual token from Twilio Console

Step 2: Restart Server
  → npm start

Step 3: Verify Configuration
  → curl http://localhost:3000/api/notifications/status
  → Should show Twilio configured: true
