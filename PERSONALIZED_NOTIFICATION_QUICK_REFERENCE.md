# Personalized Notification - Quick Reference

## What's New

✨ **Personalized test notifications in menstruation screen**

## Location

```
Menstruation Screen
    ↓
Top Navigation Bar (Right side)
    ↓
Notification Icon (🔔)
    ↓
Dialog → Select "Personalized"
```

## How to Use (3 Steps)

### Step 1: Click Notification Icon
- Top right of menstruation screen
- Bell icon (🔔)

### Step 2: Select "Personalized"
- Dialog shows 4 options
- Tap "Personalized" with heart icon
- Subtitle: "Based on your health data"

### Step 3: Notification Sent
- Local notification appears
- WhatsApp message sent (if phone saved)
- Success message shown

## Messages by Cycle Phase

| Phase | Days | Message | Emoji |
|-------|------|---------|-------|

| Follicular | 6-13 | Energy is rising! | 🌼 |
| Ovulation | 14-16 | Peak energy and confidence! | 🔥 |
| Luteal | 17+ | Time for self-care | 🌙 |

## Requirements

- ✅ At least one menstruation log
- ✅ Phone number saved (for WhatsApp)
- ✅ Internet connection

## Testing

```
1. Open Menstruation Screen
2. Click notification icon (🔔)
3. Select "Personalized"
4. Verify notification appears
5. Check message matches cycle phase
6. Check WhatsApp (if phone saved)
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| No notification | Check internet |
| WhatsApp not received | Add phone number |
| Wrong message | Check cycle day |
| Error | Restart app |

## Behind the Scenes

1. Gets user's menstruation logs
2. Calculates current cycle day
3. Determines cycle phase
4. Generates personalized message
5. Sends local notification
6. Sends WhatsApp (if phone saved)
7. Shows success message

## Integration

- Uses existing notification system
- Integrates with phone number feature
- Works with Twilio WhatsApp
- Uses user's cycle data
