# Theme & Language - Quick Start Guide

## What's New ✨

Your app now has:
- 🌓 Light & Dark Themes
- 🌐 English & Hindi Support
- ⚙️ Settings Panel in Profile

## How to Use

### Access Settings
1. Open the app
2. Go to **Profile** screen
3. Click the **⚙️ Settings** button (top right)
4. Choose your preferences

### Change Theme
1. Open Settings
2. Select **Theme**
3. Choose:
   - 💡 Light Mode (default)
   - 🌙 Dark Mode
4. Changes apply instantly

### Change Language
1. Open Settings
2. Select **Language**
3. Choose:
   - 🇬🇧 English
   - 🇮🇳 हिंदी (Hindi)
4. Entire app updates immediately

## Troubleshooting

### Settings button not visible?
- Make sure you're on the Profile screen
- Look for ⚙️ icon in the top right

### Language not changing?
- Make sure you selected a language
- Close and reopen the app

### Theme not changing?
- Make sure you selected a theme
- Try closing and reopening the app
